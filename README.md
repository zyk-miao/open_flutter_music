苦于开一个音乐app会员无法满足听歌需求，遂有了自写app的想法，通过自己上传来管理曲库，经历大半年陆陆续续的边自学边写，终于搞出来了一个自己觉得将就可以用的app。
它使用springboot作为后端,[后端项目](https://github.com/zyk-miao/open_flutter_music_back.git),通过上传音乐文件到对象存储以及从油管获取音频。
